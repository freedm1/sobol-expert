# Соболь — база знаний

Проект: сбор и структурирование информации об эксплуатации и ремонте ГАЗ Соболь.

## Двухуровневая структура

```
INDEX.md              ← всегда читай первым. Краткие факты по всем темам.
                        Каждая запись содержит ссылку на детальный файл.

kb/
  engine/             — двигатели (ЗМЗ-402, ЗМЗ-405, ЗМЗ-406, дизель)
  brakes/             — тормозная система
  suspension/         — подвеска, рулевое управление
  electrics/          — электрика, проводка
  transmission/       — КПП, мосты, сцепление
  body/               — кузов, салон, остекление
  operation/          — общая эксплуатация, ТО, расходники
```

**INDEX.md** — краткие факты (цифры, артикулы, ключевые нюансы). Влезает в контекст целиком.
**kb/\*/\*.md** — детали: диагностика, пошаговое описание, типичные ошибки. Читаются по необходимости.

## Правило обновления

Каждая новая тема → два действия:
1. Создать/дополнить `kb/[категория]/[тема].md` с деталями
2. Добавить/обновить 2–4 строки в `INDEX.md` с ссылкой на этот файл

## Основные источники

- **zr.ru** — журнал За Рулём
- **avto.pro** — авто-форум
- **gazelinfo.ru** — статьи по ремонту

## Команды

- `/project:collect [тема]` — найти информацию в интернете и сохранить в базу
- `/project:ask [вопрос]` — найти ответ в существующей базе (начинает с INDEX.md)
