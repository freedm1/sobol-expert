# ШРУС и крестовины переднего привода — Соболь 4x4

> Применимость: Соболь 4×4 (полный привод)
> Модели: Соболь 2217 4×4, 2752 4×4

## Конструкция

На Соболе 4×4 передний привод выполнен через **карданные крестовины** (а не классические ШРУСы-«гранаты» легкового типа). Каждая полуось имеет по 2 крестовины.

Каждая сторона — 3 сальника.

## Артикулы

| Деталь | Артикул |
|---|---|
| Крестовина привода переднего моста | **53А-2201025** / **53-2201800** |
| Крестовина «Сибдеталь» (аналог) | 35×93 мм (размер стакана) |
| Комплект крестовины (91423201) | с стопорными кольцами |
| Сальники полуоси | 52×72 мм (6 шт. на оба привода) |

## Симптомы

| Симптом | Причина |
|---|---|
| Хруст при включении 4WD и повороте | Изношена крестовина |
| Хруст при начале движения на 4WD | Крестовина с игольчатым подшипником сломана |
| Вибрация при движении на 4WD | Дисбаланс или люфт крестовины |
| Течь масла из-под пыльника | Разрушен пыльник → абразив внутри → ускоренный износ |

## Диагностика

1. Включить 4WD, проехать на малой скорости с поворотом руля до упора
2. Хруст при полном повороте влево = левая крестовина; вправо = правая
3. Поднять переднее колесо, покрутить рукой — ощутимый люфт = крестовина изношена

## Замена крестовин

**Инструмент:** домкрат, головки 12, 19 мм, выколотка, молоток, съёмник для стопорных колец, шприц со смазкой.

1. Снять колесо
2. Снять тормозной суппорт, подвесить на проволоке (не на шланге)
3. Снять тормозной диск
4. Открутить болты ступицы (4 болта, головка 19 мм)
5. Извлечь привод из переднего моста (тянуть на себя, помогая монтажкой)
6. Снять стопорные кольца с крестовины (съёмник)
7. Выбить стаканы крестовины выколоткой — стаканы снимаются с характерным «чпоком»
8. Установить новые крестовины, запрессовать стаканы
9. Установить стопорные кольца
10. При необходимости заменить сальники (3 шт. на сторону: 52×72 мм)
11. **Набить крестовины смазкой №158** (специальная для шарниров)
12. Собрать в обратном порядке

**Время:** 3–4 часа на оба привода.

## Важные нюансы

- **Комплект может не включать** стопорные пластины болтов ступицы — купить отдельно (без них болты откручиваются)
- После сборки — проверить затяжку болтов ступицы через 500 км
- При разборке удобно пометить положение деталей — при сборке не перепутать
- Если в стаканах крестовины **сломаны иголки** — менять только крестовину; стаканы без иголок повторно использовать нельзя
- Смазка: **Литол-24** не подходит — только специальная смазка для крестовин №158 или CV-joint grease

## Ресурс и обслуживание

Ресурс крестовин: **60–120 тыс. км** при езде по асфальту, **30–50 тыс. км** при активном использовании 4WD на грунтовках.

Продление ресурса:
- Не использовать 4WD на сухом асфальте (нет межосевого диффа → перегрузка крестовин)
- Проверять целостность пыльников при каждом ТО
- Менять крестовины при первых признаках хруста — запущенный износ разрушает стаканы привода

## Типичные ошибки

**Использовать Литол-24 вместо №158** — Литол смывается, крестовины сухие → ресурс 10–20 тыс. км.

**Не менять сальники при замене крестовин** — если они потеют, грязь снова попадёт в крестовины.

**Затянуть без стопорных пластин** — болты ступицы откручиваются сами.
