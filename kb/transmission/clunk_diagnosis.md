# Стук при трогании — диагностическое дерево

> Применимость: все двигатели
> Модели: Соболь 2217, 2752, 2310 — все

## Характер стука — первичная классификация

Прежде всего определить: **когда именно стучит?**

| Когда | Куда смотреть |
|---|---|
| Только при трогании вперёд | Карданный вал, задний мост |
| Только при трогании назад | То же + люфт в КПП |
| При трогании в обе стороны | Сцепление, крестовины, дифференциал |
| При сбросе/наборе газа на любой скорости | Крестовины кардана |
| При повороте на 4WD + трогание | ШРУС переднего моста |
| Стук + вибрация в пол | Дисбаланс кардана, промопора |

## Дерево диагностики

```
СТУК ПРИ ТРОГАНИИ
│
├── Слышен сзади?
│   ├── ДА → задний мост / кардан
│   │   ├── Тест: нейтраль на 40 км/ч — стук пропал?
│   │   │   ├── ДА → значит нагрузка нужна → кардан или редуктор
│   │   │   └── НЕТ → подшипники полуосей или барабан
│   │   │
│   │   ├── Раскачать кардан руками: люфт?
│   │   │   ├── ДА → крестовины кардана (→ замена)
│   │   │   └── НЕТ → редуктор или полуоси
│   │   │
│   │   └── Полуось: подёргать вбок — люфт в кулаке редуктора?
│   │       ├── ДА → подшипник дифференциала или полуоси
│   │       └── НЕТ → редуктор
│   │
├── Слышен спереди / по центру?
│   ├── Подёргать кардан руками (под машиной): люфт?
│   │   ├── ДА → крестовина или фланец
│   │   └── НЕТ → сцепление / КПП
│   │
│   ├── Стук с характерным щелчком «раз» при каждом трогании?
│   │   └── Скорее всего — подушки двигателя (→ проверить, затянуть)
│   │
│   └── Стук при отпускании сцепления?
│       ├── ДА → выжимной подшипник или диск сцепления
│       └── НЕТ → КПП (фиксаторы вилок)
│
└── Соболь 4x4: стук только в режиме 4WD?
    └── ШРУС переднего моста (крестовины) → хруст/стук → замена
```

## Подробно по каждому узлу

### Крестовины карданного вала
**Симптом:** одиночный стук при трогании вперёд или назад, затем тишина.
**Проверка:** вылезти под машину, обеими руками взяться за кардан и попробовать раскачать его — люфт в крестовинах ощущается как ход без сопротивления.
**Решение:** замена крестовин. Не затягивать — лопнувшая крестовина пробивает пол кабины.
→ [Детали](kb/transmission/driveshaft.md)

### Редуктор заднего моста
**Симптом:** стук при трогании, гул при движении, иногда вибрация.
**Тест:** разогнаться до 60 км/ч → нейтраль → стук/гул пропал? → это редуктор.
**Проверка:** снять крышку редуктора, осмотреть шестерни и подшипники.
→ [Детали](kb/transmission/rear_axle.md)

### Подушки двигателя
**Симптом:** толчок / удар при трогании, особенно при резком старте. Двигатель видно как он «ныряет» при газовании.
**Проверка:** попросить помощника резко нажать газ при нейтрали — видеть смещение двигателя больше нормы (> 5–7 мм).
**Решение:** замена подушек. Затягивать болты только после запуска двигателя.
→ [Детали](kb/engine/engine_mounts.md)

### Сцепление
**Симптом:** стук или вибрация в момент отпускания педали сцепления.
**Проверка:** нажать сцепление — тишина; отпустить — стук. Причины: изношен диск (неравномерный износ), люфт в корзине, изношен выжимной подшипник.
→ [Детали](kb/transmission/clutch_replacement.md)

### ШРУС переднего моста (Соболь 4x4)
**Симптом:** хруст/стук при трогании в 4WD, особенно на повороте.
**Проверка:** включить 4WD, повернуть руль до упора, тронуться — хруст = ШРУС.
→ [Детали](kb/transmission/front_cv_joint.md)

### Промежуточная опора кардана
**Симптом:** стук + вибрация по кузову при трогании, особенно на ровном асфальте.
**Проверка:** потрогать рукой резиновую подушку опоры — трещины, размягчение. Покачать опору — люфт.
**Решение:** замена резиновой подушки опоры (подшипника в сборе).

## Нюансы Соболя

- На Соболе 4x4 стук при трогании в 2WD и в 4WD — разные причины: в 2WD исключить передний мост
- Новые подушки двигателя дают неравномерный стук первые 100–200 км — норма пока «осядут»
- Крестовины закисают в обойме — при замене обязательно шприцевать маслёнки
- Старые Соболи (до 2000 г.): стук при трогании с горки + задний ход → изношены полуосевые шестерни дифференциала
