# Sobol Expert — база знаний по ГАЗ Соболь

Справочник по эксплуатации и ремонту автомобилей ГАЗ Соболь (2217, 2752, 2310) с двигателями ЗМЗ-402, ЗМЗ-405, ЗМЗ-406 и УМЗ-2.7.

Собран на основе форумов, бортжурналов и практического опыта владельцев.

## Структура

```
INDEX.md        — краткий справочник по всем темам (цифры, артикулы, ключевые нюансы)
kb/
  engine/       — двигатели: масло, охлаждение, ГРМ, топливная, зажигание, датчики
  brakes/       — тормозная система
  suspension/   — подвеска и рулевое управление
  electrics/    — электрика и проводка
  transmission/ — КПП, мосты, сцепление, кардан
  body/         — кузов, остекление, антикор
  operation/    — ТО, эксплуатация, расходники
```

`INDEX.md` — точка входа. Содержит краткие факты по каждой теме и ссылки на детальные файлы в `kb/`.

## Охват

100 тем, 112 файлов. Основные разделы:

- Двигатели ЗМЗ-402/405/406 — масло, свечи, ремни, цепи ГРМ, термостат, помпа, прокладки
- Датчики инжектора — ДМРВ, ДТОЖ, ДПКВ, ДПРВ, лямбда-зонд, датчик давления масла
- Тормоза — колодки, диски, цилиндры, прокачка, регулятор давления (колдун)
- Подвеска — шаровые, сайлентблоки, рессоры, амортизаторы, рулевые тяги
- Трансмиссия — КПП (GL-4!), мосты, сцепление, карданный вал, раздатка, ШРУС 4×4
- Электрика — генератор, стартер, АКБ, предохранители, OBD-диагностика
- Кузов — антикор, уплотнители, лобовое стекло, шумоизоляция
- Эксплуатация — ТО, подготовка к зиме/лету, покупка б/у, аварийный запас

## Ключевые нюансы Соболя

- **КПП: только GL-4** — GL-5 разрушает синхронизаторы
- **ДПКВ**: самый внезапный отказ, возить запасной (Bosch 0261210302)
- **Муфта вентилятора**: болт левая резьба — откручивать по часовой стрелке
- **Цепь ГРМ**: менять только комплектом (обе цепи + оба натяжителя + башмаки)
- **Прокачка тормозов**: задние колёса должны быть вывешены (колдун!)
- **Подушки двигателя**: затягивать только после запуска двигателя
